## Installation

1. Clone the repo:
```bash
git clone https://github.com/lakshraina2/CUCHD-Portal-Checker.git
cd CUCHD-Portal-Checker
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the GUI:
```bash
python CUIMS_gui.py
```

## Dependencies

- `tkinter`
- `selenium`
- `webdriver_manager`
- `beautifulsoup4`
- `pillow`
- `pandas`
